# ReadyTalk Pairing Station Setup
Setup scripts for ReadyTalk engineering pairing stations that installs (not a complete list):

* Bash aliases
* Google Chrome
* Java JDK
* VirtualBox
* Docker
* Sublime
* IntelliJ Idea
* Atom
* NVM

## OSX Setup

    ./setup-osx.sh

This script can safely be run more than once.
